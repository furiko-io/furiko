# Contribution Guide

Thanks for wanting to contribute to Furiko! There are lots of ways to start contributing.

Check out the [Contributing](https://furiko.io/guide/contributing/overview/) section on the website, which describes high-level tips and guidelines for contributing to the entire Furiko project.

## Development Guide

To get information on how to start developing and contributing specifically to this repository (`furiko-io/furiko`), see the [Development Guide](./DEVELOPMENT.md) for a comprehensive guide on doing so.

## Submitting a Pull Request

We use GitHub for issue tracking and collaborative version control. For more information about how to submit a pull request, you can refer to [GitHub's guides](https://docs.github.com/en/pull-requests).
